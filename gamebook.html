Gamebook

Retro text interface. Type commands in the box and press Enter.


        

          
          
        

      

      

        
How to Use:

        
    
          
  • Navigate the gamebook by selecting numbered options.
    • 
          
  • If you want to create or modify a gamebook, type edit to use editor mode.
    • 
          
  • When you have edited a gamebook, type show to get the nodes you can copy-paste to a savefile
    • 
          
  • If you have copy-pasted into a savefile, type load to copy-paste back into the gamebook system
    •
